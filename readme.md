# Pokedex

Esse sistema função de buscar as informações dos pokemons através da API do [`PokeAPI`](https://pokeapi.co/).

<b style="color: red;">Atenção:</b> Esse projeto que criei é com o intuito de estudar programação através da linguagem Javascript.

## Usando a Pokedex

No momento atual os requests trazem apenas as sprites e o Nome do Pokemon.

![Menu Principal](/assets/img/main_page.png "Menu Principal")

A usabilidade é muito simples, coloque o nome do Pokemon e irá trazer os dados do pokemon, como visto a seguir:

![Usabilidade](/assets/img/pokemon.png "Usabilidade")

## Requisitos

Atualmente só há dois requisitos:
[`NodeJS`](https://nodejs.org/en)
[`ViteJS`](https://vitejs.dev/)
